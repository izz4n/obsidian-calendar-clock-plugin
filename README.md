# Calendar Clock Companion for Obsidian

An elegant, zero-configuration companion plugin that displays a live, ticking clock directly above Liam Cain's **Calendar** plugin in your sidebar workspace.

## Features

- **Theme Native:** Adapts seamlessly to light and dark theme modifications.
- **Configurable Formats:** Easily toggle between 12-hour (AM/PM) and 24-hour clocks.
- **Optional Seconds Track:** Hide or show seconds to customize visual focus.
- **Wiggle-Free:** Utilizes monospace layout fonts so time changes do not shift your sidebar panels.

## Installation

### Manual Installation
1. Download `main.js` and `manifest.json` from the latest release.
2. Navigate to your vault's plugin directory: `.obsidian/plugins/`
3. Create a folder named `calendar-clock` and move both files inside.
4. Reload Obsidian and toggle the plugin on!
